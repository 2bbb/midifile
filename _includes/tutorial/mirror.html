Mirror a MIDI file in time or pitch. The following example program will perform a mirror operation on a MIDI file, either reversing the timeline of the file, or mirroring the pitches in the file. Without options, the time of the MIDI file will be reversed. Specifying the -p option with a MIDI key number to reflect on will cause the pitches to be mirrored, such as -p 60 to invert the pitch around middle-C. To both mirror time and pitch, add the -r option, such as `mirror -r -p 60 input.mid output.mid` or `mirror -rp 60 input.mid output.mid`. As a demonstration, here are four MIDI files for Bach's Well-tempered Clavier, Book II, prelude 2 in C minor in for orientations:

  2. Retrograde
  3. Inverted around middle-C
  4. Retrograde Inversion
