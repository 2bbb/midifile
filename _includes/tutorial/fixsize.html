Fix track byte counts MIDI track chunks byte counts are often incorrect in Standard MIDI files. This example program shows how to correct the byte counts by reading the MIDI file and then writing it out again.

The MidiFile class ignores the byte-count values when reading a MIDI file, so incorret byte counts do not affect it. When writing a file out again, the byte counts is calculated and automatically corrected.

The MidiFile class automatically expands running-status messages, so the size of the MIDI file may increase if running-status was originally used. ``` cpp {% include tutorial/code/fixsize.cpp %} ``` Functions used in this example:

MidiFile::read — Read in a Standard MIDI File.
MidiFile::write — Write a Standard MIDI File.
